# Kodilla Library

## How to run locally

Clone repository:

`git clone https://github.com/bartlomiejlorenowicz/Kodilla-Library.git`
```bash 
 
cd Kodilla-Library
```
Start database:
```Bash

   docker compose up -d
```
   
Run application:

```Bash

cd library
```

```Bash

./gradlew bootRun
```
   

Application runs on:
http://localhost:8080
